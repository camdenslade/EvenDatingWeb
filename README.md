# Even Dating Website

A modern multi-page website for Even Dating built with React 18, TypeScript, Vite, and React Router.

## Features

- **Home Page**: Landing page with app branding and primary actions
- **Roadmap Page**: Product development timeline and feature plans
- **Support Page**: FAQ, contact options, and resources
- Dark mode theme matching the app design
- Responsive design for all screen sizes
- Clean, minimalist UI with subtle background shapes
- TypeScript for type safety
- React Router for navigation

## Getting Started

### Install Dependencies

```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

Build for production:

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
